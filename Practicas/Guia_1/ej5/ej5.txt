5. Convertir las siguientes expresiones lógicas en Pascal al lenguaje C:
a) ( z < 5 ) AND ( ( a < = 5 ) OR ( c > 0 ) )
b) ( z = CHR( 0 ) ) OR NOT( total = 0 ) 


RESPUESTAS:

a)

( z < 5 ) && ( ( a <=5 ) || ( c > 0 ) )

-------------

En Pascal, CHR(n) devuelve el carácter cuyo código ASCII es n

Entonces CHR(0) equivale al codigo ASCII del caracter nulo en C, que es \0

Entonces:

-------------

b)

( z == '\0' ) || !( total == 0 )
